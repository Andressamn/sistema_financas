# App de Finanças Pessoais

Aplicação full stack para controle financeiro pessoal.

## Stack
- **Frontend:** React + Vite + TypeScript + Tailwind CSS + Chart.js
- **Backend:** Node.js + Express + TypeScript + Prisma ORM
- **Banco:** PostgreSQL
- **Auth:** JWT + bcrypt

## Estrutura
```
App_finanças/
├── frontend/   # SPA React
├── backend/    # API REST
└── README.md
```

## Funcionalidades (MVP)
- [ ] Cadastro e login de usuário (JWT)
- [ ] CRUD de transações (receitas/despesas)
- [ ] Categorias customizáveis
- [ ] Dashboard com gráficos
- [ ] Metas de economia

## Pré-requisitos
- Node.js LTS (https://nodejs.org)
- PostgreSQL (https://www.postgresql.org/download/)
- Git
